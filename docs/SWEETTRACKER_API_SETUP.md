# 🔑 스위트트래커 API 설정 가이드

## 1. API Key 발급

### 스위트트래커 회원가입
1. https://tracking.sweettracker.co.kr/ 접속
2. 회원가입 (개인/사업자 선택 가능)
3. 이메일 인증 완료

### API 신청
1. 로그인 후 "API 신청하기" 클릭
2. 플랜 선택:
   - **무료**: 1일 100건 (테스트용)
   - **베이직**: 월 5,500원 (1일 500건)
   - **스탠다드**: 월 16,500원 (1일 2,000건)
   - **프리미엄**: 월 33,000원 (1일 5,000건)
3. 약관 동의 및 신청
4. 승인 후 API Key 발급 완료

---

## 2. 시스템 설정

### 웹 관리자 페이지에서 설정
1. **관리자 페이지** 로그인
2. 좌측 메뉴에서 **⚙️ 설정** 클릭
3. "배송 추적 API 설정" 섹션에서 API Key 입력
4. **[저장]** 버튼 클릭
5. **[테스트]** 버튼으로 연결 확인

---

## 3. 사용 방법

### 자동 기능 (설정 완료 시)
- ✅ 운송장 번호 클릭 → 배송 조회 페이지로 이동
- ✅ API를 통한 실시간 배송 정보 조회 가능

### 배송 조회
```javascript
// 프론트엔드에서 사용 예시
const response = await axios.get(
  `${API_BASE}/tracking/track/${trackingNumber}`, 
  { params: { courier: 'CJ대한통운' } }
);
```

### 지원 택배사
- CJ대한통운
- 로젠택배
- 우체국택배
- 한진택배
- 롯데택배
- 기타 100+ 택배사

---

## 4. 문제 해결

### API Key 오류
- API Key가 정확한지 확인
- 스위트트래커 사이트에서 Key 재확인
- 플랜 만료 여부 확인

### 배송 조회 실패
- 운송장 번호가 정확한지 확인
- 택배사 정보가 올바른지 확인
- API 할당량 초과 여부 확인 (무료: 100건/일)
